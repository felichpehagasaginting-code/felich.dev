## 🛡️ The Future-Proof Stack: Next.js 16 & React 19

Innovation never stops. To ensure this portfolio remains at the absolute cutting edge, I've implemented a comprehensive migration to **Next.js 16** and **React 19**. This wasn't just a version bump; it was a fundamental architectural update.

### ⚡ The Next.js 16 Shift
Next.js 16 brings massive performance improvements via **Turbopack**, but it also introduces stricter rules for Server Components. I've abstracted all client-side-only components (like the Command Palette and 3D Visualizers) into a specialized `DynamicClientComponents` wrapper. This ensures full compatibility with the new server-side rendering constraints while maintaining a 100% lighthouse score.

### 📦 React 19 & Async Params
With **React 19**, the ecosystem is moving towards a more asynchronous future. Next.js 15+ introduced **Async Params**, meaning route parameters are now promises. I've refactored the entire blog and project engines to `await` these params, ensuring that data fetching remains type-safe and non-blocking.

## 🧠 The AI Assistant: Why Gemini 2.0 Flash?

If you look at the bottom-left corner, you'll see **Felich AI**—a highly intelligent assistant built directly into this site. It doesn't just read static text; it can parse Markdown, render code blocks, and even **speak to you using Text-to-Speech (TTS)**.

Many would ask: *"Why not use GPT-4?"*

Because **Gemini 2.0 Flash** is undeniably superior for this specific use case. It provides incredibly fast streaming responses with virtually zero latency. The integration with Google's ecosystem and the superior context window management ensures the bot never hallucinates or loses track of its role.

<HighlightBox type="info">
The performance difference is night and day. With Next.js 16 and Gemini 2.0, the time-to-first-token is under 200ms, providing a truly interactive experience.
</HighlightBox>

## 🛠️ The SVG Overhaul & PWA
Initially, I used a CDN for technology logos. It was a bottleneck. I migrated every single logo into a highly optimized, local inline SVG library (`SkillIcons.tsx`). 

- **Zero external network requests.**
- **Perfect offline support.**
- **Instant rendering.**

Add in the dynamic **Progressive Web App (PWA)** integration, and this site installs directly on your device as a native app—something a basic SPA simply cannot match.
