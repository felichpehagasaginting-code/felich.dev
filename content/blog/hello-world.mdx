Welcome to my very first blog post! In this article, I want to share my journey of building a completely custom and highly interactive portfolio website.

## Why Build from Scratch?

There are countless templates available online, but as a Software Engineer, I wanted my portfolio to be a true representation of my skills. Building from scratch allowed me to:

1.  **Fully control the architecture**: Using Next.js App Router for optimal performance.
2.  **Experiment with micro-interactions**: Implementing custom cursors and smooth page transitions with Framer Motion.
3.  **Integrate 3D Elements**: Using \`@react-three/fiber\` to add a unique flavor to the hero section.

## The Tech Stack

-   **Framework**: Next.js 14
-   **Styling**: Tailwind CSS
-   **Animations**: Framer Motion
-   **3D Rendering**: Three.js & React Three Fiber
-   **Backend**: Express.js (for Guestbook & Contact APIs)

### The Guestbook

One of my favorite features is the real-time guestbook. Instead of relying on a heavy third-party database, I built a lightweight JSON-backed storage system using Node.js \`fs\` module. It ensures the site remains fast while allowing visitors to leave permanent remarks.

## Conclusion

Building this portfolio was an incredibly rewarding experience. It allowed me to push the boundaries of frontend engineering and design. If you have any feedback or just want to say hi, feel free to sign the Guestbook!
